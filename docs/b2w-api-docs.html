1 Introdução

A API possibilita integrar sistemas externos com a nossa plataforma B2B Online. A API permite consultar produtos de um catálogo, inserir pedidos e consultar valor de frete. Toda a nossa API é exposta usando o padrão REST através do protocolo HTTP e sua segurança é garantida através do protocolo SSL e o método de autenticação BASIC AUTHENTICATION do padrão HTTP. Os métodos HTTP usados variam de acordo com a função sendo utilizada. Segue a tabela de métodos versus ação usada.

AçãoMétodo
ConsultaGET
InserçãoPOST
AtualizaçãoPUT
RemoçãoDELETE
